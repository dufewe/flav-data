# 衰变模式缩写与观测量命名

| B⁰ → K*⁰e⁺e⁻ | B02Kstee | |
| B⁰ₛ → φμ⁺μ⁻ | Bs0.2.phimumumu | φ → K⁺K⁻ |

## 观测量类型

### CP 平均角分布观测量 (B → K*ℓℓ)

| 符号 | 全称 | 取值范围 |
|------|------|---------|
| FL | 纵向极化分数 | [0, 1] |
| S3 | T-odd 角系数 | [-0.5, 0.5] |
| S4 | 角系数 | [-1, 1] |
| S5 | 角系数 | [-1, 1] |
| AFB_CP | CP 平均前后不对称性 | [-1, 1] |
| S7 | T-odd 角系数 | [-1, 1] |
| S8 | T-odd 角系数 | [-1, 1] |
| S9 | T-odd 角系数 | [-0.5, 0.5] |

### CP 不对称观测量

| 符号 | 全称 |
|------|------|
| A3 | S3 的 CP 不对称 |
| A4 | S4 的 CP 不对称 |
| A5 | S5 的 CP 不对称 |
| A6s | S6 CP 不对称 |
| A7 | S7 的 CP 不对称 |
| A8 | S8 的 CP 不对称 |
| A9 | S9 的 CP 不对称 |

### 优化观测量 (P 观测量)

| 符号 | 全称 |
|------|------|
| P1 | = S3 / (1 - FL) |
| P2 | = β/(2(1-FL)) 部分约定 |
| P3 | = Im(G⊥G∥*)/(1-FL) 部分约定 |
| P'4 | 优化 S4 |
| P'5 | 优化 S5 |
| P'6 | 优化 S6 |
| P'8 | 优化 S8 |

### Bs → φℓℓ 观测量

| 符号 | 全称 |
|------|------|
| FL | 纵向极化分数 |
| A'6 | 约化前后不对称 = A6/(1-FL) |
| S3 | 角系数 |
| A9 | CP-odd 角系数 |

### B → Kℓℓ 观测量

| 符号 | 全称 |
|------|------|
| FH | 角分布平坦项 |
| AFB | 前后不对称 |

### 分支比

| 类型 | 单位 | 说明 |
|------|------|------|
| dBr/dq² | 10⁻⁸ GeV⁻² | 微分分支比 |
| Br (相对) | 10⁻⁵ | 相对归一化道 |

### Lepton Flavor Universality Ratios

| 符号 | 定义 |
|------|------|
| RK | Γ(B→Kμμ) / Γ(B→Kee) |
| RK* | Γ(B→K*μμ) / Γ(B→K*ee) |
| Rh | Γ(B→hμμ) / Γ(B→hee) (h=π,K,...) |

## LaTeX 表示

```python
OBSERVABLE_LATEX = {
    # CP-averaged
    'FL': '$F_L$',
    'S3': '$S_3$', 'S4': '$S_4$', 'S5': '$S_5$',
    'S7': '$S_7$', 'S8': '$S_8$', 'S9': '$S_9$',
    'AFB_CP': '$A_{\\rm FB}^{\\rm CP}$',

    # CP-asymmetric
    'A3': '$A_3$', 'A4': '$A_4$', 'A5': '$A_5$',
    'A6s': '$A_{6s}$', 'A7': '$A_7$', 'A8': '$A_8$', 'A9': '$A_9$',

    # Optimised
    'P1': '$P_1$', 'P2': '$P_2$', 'P3': '$P_3$',
    "P4'": "$P_4'$", "P5'": "$P_5'$", "P6'": "$P_6'$", "P8'": "$P_8'$",

    # Bs->phi ll
    "A6'": "$A'_6$",

    # B->K ll
    'FH': '$F_H$',
    'AFB': '$A_{\\rm FB}$',

    # LFU ratios
    'RK': '$R_K$',
    'RKst': '$R_{K^*}$',
}
```

## 新观测量添加规则

当论文中出现本文件中没有的新观测量时：
1. 确认观测量的物理含义和标准符号
2. 添加到对应分类的表格中
3. 在 LaTeX 映射表中添加对应条目
4. 如果是新的衰变模式，在衰变模式缩写表中添加
